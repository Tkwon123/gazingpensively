# Gazing Pensively

Front-end assets for www.gazingpensively.club.

## Architecture
* Firebase for hosting and persistent data
* WikiQuotes for inspiration
* Jeff Burlin picture via Jeff BUrlin

## Local builds
Deploying local server:
```
npm install && bower install
gulp serve
```
